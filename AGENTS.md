# Turco Manager AI Rules

Project type:
- Football Manager Simulation

Stack:
- Backend: Node.js + Express + MongoDB
- Frontend: React + Vite

Important systems:
- JWT Authentication
- Team morale
- Team chemistry
- Team fatigue
- Team form
- Match simulation
- League standings

Rules:
- Never break auth system
- Keep API routes compatible
- Keep frontend and backend synchronized
- Use clean modular code
- Prefer reusable React components
- Protect manager-only routes
